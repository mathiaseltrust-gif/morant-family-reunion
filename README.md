# Morant Family Bloodline Registry

Static GitHub Pages starter site for morantfamilyreunion.com.

## Publish with GitHub Pages

1. Create a new GitHub repository named `morant-family-reunion`.
2. Upload `index.html` and `styles.css`.
3. Go to Settings → Pages.
4. Source: Deploy from branch.
5. Branch: main / root.
6. Add custom domain: `morantfamilyreunion.com`.
7. In GoDaddy DNS, add the GitHub Pages records.

## Form setup

Replace this line in index.html:

https://formspree.io/f/REPLACE_WITH_YOUR_FORM_ID

with a real Formspree endpoint, Google Form link, or Supabase form later.
